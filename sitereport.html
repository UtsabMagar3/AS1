Overview

This website provides information on the entire web development learning process. During my web development adventure, I learnt a variety of coding methodologies, design and debugging processes and more.


Learning HTML and CSS

HTML stands for Hypertext Markup Language, which is used to create the basic structure of a website and provides an overview of it. Similarly, CSS stands for Cascading Style Sheets, which are utilized to style the basic HTML structure and make our website more appealing.

Throughout my learning experience, I learned a lot about HTML, CSS, various types of tags, elements, and CSS properties. In my portfolio website, I used HTML5 and CSS3.


Learning Git and Github

Git is a distributed version control system (VCS) that allows developers to track changes to their codebase, whereas GitHub is a cloud-based platform that hosts Git repositories and offers extra capabilities such as collaboration, code review, and project management. Although they sound similar, their functions and applications are quite distinct.

I learned different git commands for uploading different files to Github. Git clone, git push, git pull, and git commit are the most basic commands I learned and utilized over my learning journey. Aside from those commands, I learnt about repositories, branches, ways to create repositories, branches, and files on github, and more.


Design layout

I wanted my portfolio to look more professional, therefore I chose minimal and straightforward colors. Different transition and hover effects have been employed to improve the website's appearance.My portfolio website consists of a header and section parts.The anchor tag was used to link various pages. The header section includes a navigation bar with a logo and links to different pages.

Navigation bar

A navigation bar, often known as navbar, is a user interface element seen in graphical user interfaces (GUIs) including websites and applications. It is often located along the top or side of a webpage or screen and functions as a menu or guide to assist users in navigating the site or app's many parts or pages.

My portfolio features a navbar at the top of the page. The navbar contains a variety of contents, including a logo and a page link. In terms of navigation bar design, I used div tags to create two divisions, with the logo on the left and different page links on the right. I display such divisions using the flex attribute. Hover effect has also been added to the page links.


Home page

The homepage of a website is the main or initial page that people usually see when they visit the site's root domain. My home page features a variety of content, including an image, an introduction, and links to social media platforms such as Facebook, Twitter, and LinkedIn. Different types of effects have been employed on my site to make my portfolio more visually appealing. The homepage is as simple as possible so that visitor may grasp it.

In terms of the design approach, I employed several HTML tags and CSS attributes. I separated my homepage into different division and displayed them using the flex feature. Other CSS attributes like padding, margin, and shadow were also employed. I've added an opacity reduction effect to the photo while hovering it.

Responsive design

Responsive design is an essential component of modern web development that ensures websites adjust easily to various screen sizes, resolutions, and device capabilities. Responsive design helps to improve the website's performance and user experience. Adapting adaptable design allows websites to function across multiple devices. The viewport meta tag was used to create responsive design, and media queries were also employed.


Challenges

I encountered numerous issues during the website's construction phase. Some of the difficulties included browser compatibility, responsive design issues, and content management issues.Even though it was difficult at first, I was able to overcome these challenges by conducting study on various internet platforms. I gathered several references online. All of these issues enabled me to learn and experience fresh vital lessons.

But there was one issue that I couldn't overcome. When I tried to include a hamburger icon in the nav bar during mobile responsive design, it resulted in a more complex position problem. So, unfortunately, I was unable to conquer this obstacle, but I am confident that in the future, I will solve it nonetheless.
